# Warehouse Management System

A desktop Warehouse Management System built using **C# WinForms and MySQL**.

This application allows users to manage warehouse inventory including adding, updating, deleting, and viewing products.

## Features

- Add new products
- Update existing products
- Delete products
- View product inventory
- DataGridView inventory display
- MySQL database integration
- Simple inventory management workflow

## Technologies Used

- C#
- .NET WinForms
- MySQL
- Visual Studio 2022
- Git / GitHub

## Architecture

The application follows a simple layered architecture:

UI Layer (WinForms)
↓
Form Logic
↓
Service Layer
↓
Database Connection
↓
MySQL Database
